## 컨셉
> 300년 만에 부활한 마왕. 마왕성은 경매로 넘어갔고 남은 건 낡은 소환 계약서 뭉치. 랜덤 소환으로 용병을 뽑고, 시너지 편성으로 아레나 12라운드를 정복한다.

장르: **로그라이트 오토배틀러** (Super Auto Pets/TFT 계열의 싱글 플레이 변형, 브라우저 단일 HTML)

## 핵심 수치
- 런: 12라운드, 목숨 3(+1), 보스 4/8/12. 승리 후 4·8라운드에 유물 3택1.
- 경제: 라운드 수입 10 + 이자(🪙10당 1, 상한 3) + 연승(상한 3). 유닛 가격 = 등급별 2/3/4/5/6. 판매가 = 가격+(★-1)×2.
- 등급 확률: 1라운드 [85/15/0/0/0] → 12라운드 [8/18/26/30/18].
- 합성: 동일 유닛 2장 → ★+1 (최대 ★3 = 스탯 ×3).
- 운명의 소환: 기본 [40/32/18/10], 천장 2회(업그레이드 시 1회) 내 영웅+ 확정.
- 메타: 마정석 = 승수 + 클리어 8 + 8라운드 도달 2. 영구 계약 6종(10~40).

## 계약 포획 (v1.1 추가 — 포켓몬식 영입)
- 전투 승리 시(12라운드 제외) 쓰러진 적 중 최대 3종이 후보로 제시되고, **계약서를 1회 던져 영입 시도**.
- 서명 확률: 등급별 75/60/45/30/15% − (★−1)×15%p. 성공 시 **★등급 그대로** 아군 합류 (★2 적 포획 = 잭팟).
- 확률 보정: 메타 업그레이드 "구속의 인장" +15%p, 유물 "혼의 족쇄" +20%p.
- 빈 자리가 없으면 포획 불가 → "전투 전에 자리를 비워둘까"라는 새로운 의사결정.
- 연출: 계약서가 파르르 떨리는 3박자 연출 후 성공(✒️)/도주(💨) — 포켓몬 몬스터볼의 긴장감 재현.
- 설계 의도: 상점 뽑기 외 제3의 유닛 획득 경로 + "다음 적 정찰"에 '잡고 싶은 유닛 확인'이라는 두 번째 의미 부여.

## 지휘형 전투 리워크 (v1.2 — "너무 단순하다" 피드백 반영)
- **완전 자동 전투 폐지 → 지휘형 턴제**: 아군 유닛의 턴마다 플레이어가 [⚔️ 공격(대상 직접 지정)] / [✨ 스킬(쿨다운)] / [🌀 자동 행동] 중 선택. 포켓몬처럼 "잡은 몬스터에게 기술을 지시"하는 감각.
- **유닛 액티브 스킬**: 전 유닛이 쿨다운 스킬 보유 — 직업별 공용(강타/조준 사격/마력 폭발/응급 처치) + 영웅·전설급 10종은 고유 스킬(군림, 사신의 낫, 해일, 연격…).
- **마왕 스킬 레이어**: 매 턴 차오르는 마력(🔮)으로 유닛 턴과 무관하게 화염구/재생/빙결/진노 시전 — 개입의 두 번째 축.
- **자동 토글**: 🤖 버튼으로 언제든 오토 전환(AI가 스킬까지 사용). 적 부대도 동일 AI로 스킬 사용.
- **판 확장**: 전열 4 + 후열 4 + 대기석 5 (기존 3+3+4). 적도 최대 8기.
- **합성 우선 획득**: 자리가 꽉 차도 같은 유닛 ★1 보유 시 구매→즉시 합성 허용. 소환은 자리 없으면 합성 또는 골드 환전, 포획도 동일 ★ 보유 시 합성 포획 가능.
- **직업 시너지 축 추가**: 종족 5종에 더해 전사/사수/술사/지원 4직업 시너지(2/4세트) — 편성이 2축 퍼즐로.
- 신규 유닛 8종(총 33종): 고블린 새총꾼, 해골 궁수, 치유 사슴, 임프 폭격수, 바람 정령, 무덤 사제, 계약 악마, 현상금 사냥꾼.
