# 마왕상회 (가칭) — 게임 기획서 v1.0

> **한 줄 컨셉**: 용사에게 패배해 몰락한 마왕이 인간 마을에서 정체를 숨기고 잡화상을 연다. 낮에는 모험가들에게 장비를 팔고, 밤에는 그 모험가들을 의뢰에 보내 재료를 떼온다 — 그리고 내가 판 검이, 언젠가 나를 찌르러 올지도 모른다.

작성일: 2026-07-02 · 근거: `docs/game-research/` 30여 개 게임 분석 (각 시스템에 근거 표기)
플랫폼: 브라우저 단일 HTML (모바일 대응, `game.html`과 동일 스택) · 톤: **코믹 다크 판타지**

## 1. 왜 이 게임인가 (리서치 근거)
- **시장 공백**: "마왕이 장사한다" 니치에 대작이 없음 — '마왕상회'라는 작품은 실존하지 않고 유사작(마왕의 시간, 상인사가)은 전부 소규모. [06 §3]
- **검증된 훅의 힘**: 역발상 3초 훅(레전드 오브 슬라임 "몬스터가 인간을 사냥" = 글로벌 $1억). "마왕을 쓰러뜨린 용사가 내 단골손님"은 스크린샷 한 장으로 전달된다. [03]
- **검증된 코어 순환**: Recettear의 "고객=파트너"(판 장비를 모험가가 착용하고 강해짐, 50만 장+96%)와 용사식당(335만 DL)의 경영↔파견 순환. 이 공식을 정면 계승한 게임이 2007년 이후 없다. [02, 06 §3]
- **톤 선택 근거**: 이 장르 구매층은 '긴장'보다 '코지/유쾌한 파워 판타지'에 지갑을 연다 (Potionomics 27만 vs Potion Craft 100만+, "Capitalism, ho!"). 단 드라마의 무게(상실·배신)는 다크하게 유지 — 코믹 다크가 두 마리를 잡는다. [02 §요약3]

## 2. 플레이어 판타지와 아이러니 (재미의 원천)
1. **장사의 재미**: 손님 속을 읽어 적정가를 맞히는 쾌감 + 돈이 불어나는 쾌감.
2. **키우는 재미**: 어리바리 신참 모험가가 내 장비를 입고 전설이 되는 것을 지켜보는 재미.
3. **마왕의 아이러니**: 인간에게 굽신거리며 장사하는 전직 마왕이라는 코미디 + "마왕의 힘을 쓰면 편해지지만 발각된다"는 긴장 + 마지막 날의 선택(복수냐, 장사냐).

## 3. 코어 루프

### 30초 루프 — 손님 한 명
손님 입장(유형·소지금·성격 힌트 노출) → 원하는 물건 제시 → **가격 슬라이더로 값을 부름** → 손님 반응(이모지+한 줄 대사) → 성사/흥정/결렬 → 골드 + **감정(感情)이 아니라 감정(鑑定) 콤보** 적립.
- 손님 유형별 허용 마진이 다름(신참은 지갑이 얇고, 귀족은 바가지에 관대) — "고객 읽기"의 게임화. [Recettear 손님 계수]
- 기대가 ±5% 이내 적중 시 **"정가 감각" 콤보** — 연속 적중마다 보너스 배증. [Recettear Just Pin]
- 거래 결과는 **장부(자동 기록 노트)**에 쌓여 학습이 UI에 축적됨. [Moonlighter 상인 노트]

### 5분 루프 — 하루
**낮(장사)**: 진열 구성 → 손님 4~8명 응대 → 트렌드 확인("북쪽에 얼음 던전이 열렸다더라" → 화염 장비 수요↑).
**저녁(알선)**: 단골 모험가에게 의뢰 중개 — 파티 편성(리더십 캐퍼시티 내 코스트 배분)·장비 점검 후 파견. [Symphony of War 편성=덱빌딩]
**밤(정산)**: 파견 결과가 **한 줄 텍스트 로그 드라마**로 귀환(전리품/부상/영웅담) → 제작(재료→장비/물약) → 마족 참모들의 상충하는 조언 → 취침(자동 저장).
- 장사가 탐험의 퀘스트를 발주: 손님의 특별 주문("서리늑대 송곳니 단검 구해줘") → 해당 던전 의뢰가 게시판에 생성. [Dave the Diver 요구 결합]

### 세션 루프 — 1런 = 4주(28일), 30~60분
- 매주 말 **상납금 데드라인**(마을 영주에게 자릿세: 150→350→800→1,800G, 지수 성장) — 미납 시 게임 오버. 저축을 강제 회수해 인플레 차단 + 세션 목표 생성. [Recettear 주간 빚]
- 매주 **"교단의 의심" 3개 중 1개만 무마** 선택(나머지는 발동) — 적은 콘텐츠로 매주 다른 캠페인. [XCOM 2 다크 이벤트]
- 28일째 **최종 이벤트**: 용사 교단의 최종 조사 → 축적한 것(돈·단골·발각 게이지)에 따라 분기 엔딩: ①완전 은폐(장사꾼 엔딩) ②정체 폭로→단골들이 나를 지켜줌/배신함(관계 축적 판정) ③마왕성 탈환 출정(내가 키운 모험가들과 함께, 또는 그들을 상대로).

## 4. 핵심 시스템

### 4.1 상점 — 판매·제작
- **아이템**: 재료(던전 산출) → 레시피 → 완성품(무기/방어구/물약/잡화). 원료 직판 vs 가공 판매의 마진 트레이드오프. [Winkeltje]
- **레시피 숙련**: 같은 품목 반복 제작 시 품질·단가 상승 — 저가치 재료의 후반 용도. [Dave the Diver 레시피 강화]
- **수요 시스템**: 주간 트렌드(던전 소문 연동) + 같은 품목 양산 시 가치 하락(최대 -25%) — 가격 정답 고정 방지. [Travellers Rest 트렌드, Moonlighter 수요 곡선]
- **재료 일부는 부패성**(마수 고기 등) — 당일 처리 압박으로 사재기 차단. [Dave the Diver]

### 4.2 모험가 — 고객이자 파트너 (드라마 생성기)
- 생성 공식: **이름 + 종족/배경 + 불균형 트레잇 2개**("겁쟁이지만 손이 빠름") + 전적 기록. [BB, RimWorld — 06 마스터 리스트 #6]
- **내 가게에서 산 장비가 그들의 실제 전투력** — 판매가 곧 전력 투자. 단골 레벨이 오르면 더 비싼 장비를 원하고 더 어려운 의뢰를 뛼 수 있음(수요 곡선이 성장하는 단골). [Recettear, Magical Mixture Mill]
- **파견 결과**: 자동 전투(성공률 = 파티 전투력 vs 의뢰 난이도, 결과는 텍스트 로그 연출). 치명상 시 **사망 대신 "영구 흉터+휴업" 선택지** — 흉터는 트레잇으로 남아 서사가 됨. 사망 시 유품이 가게에 돌아오고 다른 단골들의 사기에 영향. [Wildermyth maiming]
- **의뢰 선택의 무게**: 보수 높은 "더러운 의뢰"(평판↓) vs 깨끗한 의뢰(평판↑) — 효율 vs 명성 트레이드오프. [Ogre Battle]

### 4.3 마왕의 힘 — 리스크 다이얼
마력을 쓰면 강력하지만 **발각 게이지(0~100)**가 오른다. [DD 횃불 다이얼의 변형]
- 예: 「심안」 손님의 적정가를 1회 훔쳐보기 / 「마정 주입」 제작품 품질 강화 / 「어둠의 가호」 파견 파티 버프 / 「위압」 흥정 강제 성사.
- 발각 게이지 구간별로 교단 조사 이벤트 강도 상승 → 최종일 판정에 직결. 마력을 아예 안 쓰면 안전하지만 가난하다 — "얼마나 마왕일 것인가"가 매일의 선택.

### 4.4 마족 참모 — 조언자 패턴 + 코미디 엔진
참모 2~3인(예: 회계 마족 리치 "돈이 곧 마력입니다", 전직 사천왕 근육 오크 점원)이 매 사건마다 **상충하는 조언** — 적은 텍스트로 개성+공략 힌트+유머 동시 전달. [King of Dragon Pass]
뉴스 티커(마을 소문)로 세계관·유머·트렌드 예고를 흘림. [Cookie Clicker 티커]

### 4.5 경제 설계
| 재화 | 소스 | 싱크 |
|---|---|---|
| 골드 (런 내) | 판매 이익, 의뢰 중개 수수료 | 재료 매입, 진열/설비, 주간 상납금(강제), 모험가 치료비 |
| 재료 | 파견 전리품, 행상인 | 제작, (부패성은 소멸) |
| 마정석 (메타) | 런 성과·업적·엔딩 보너스 | 영구 업그레이드 (아래 4.6) |
- 유지비 러닝머신: 점포 임대료(일)+참모 급여(주)가 소액 상시 압박. [BB — 단, 코지함을 위해 압박 강도는 BB보다 완만하게]
- 비용 곡선: 업그레이드 지수 성장 r≈1.15. [04 incremental 공통 수학]

### 4.6 메타 진행 — "런은 끝나도 상회는 남는다" (DD2의 실패 회피)
- **상회의 비밀 금고(마정석 트리)**: 시작 자금, 레시피 도감, 진열대 종류, 참모, 마왕 스킬 해금 — Hades식 영구 성장.
- **전설 명부(레거시)**: 런에서 전설이 됐거나 은퇴한 모험가가 명부에 등록 → 다음 런에 마정석을 내고 "단골로 재등장"(흉터·전적 유지). 회차 반복을 수집으로 전환. [Wildermyth 레거시]
- **엔딩 컬렉션**: 분기 엔딩 도감이 회차 목표.

## 5. 온보딩 (첫 5분)
점진적 공개 — 첫 화면은 어둠 속 버튼 하나: **[셔터를 연다]**. [A Dark Room]
1분: 첫 손님에게 물건 하나 판다(가격 슬라이더+이모지만) → 2분: 두 번째 손님이 특별 주문 → 3분: 첫 모험가 파견(버튼 1개) → 4분: 귀환 로그 드라마 + 제작 해금 → 5분: 참모 등장, 상납금 고지(첫 데드라인 제시). 튜토리얼 텍스트 없이 UI 해금 자체가 보상.

## 6. 밸런싱 초안 (v0.1 스프레드시트 기준값)
- 시작 골드 100 / 하루 손님 4명(주차마다 +1) / 아이템 기본가 10~500G.
- 손님 허용 마진: 신참 +5~15% · 일반 +10~30% · 귀족 +25~60% · 용사(고정가 깎음) -10~+5%.
- 상납금: 150/350/800/1,800 (합 3,100 — 28일 총수익 기대치의 ~60%를 회수).
- 파견 성공률 = clamp(35 + (파티전투력-난이도)×1.5, 5, 95)%. 대실패(사망 판정)는 실패의 25%.
- 발각 게이지: 마왕 스킬 +8~20, 주간 자연 감소 -5, 60 이상부터 조사 이벤트.

## 7. 개발 로드맵
- **v0.1 프로토타입 (다음 단계)**: 단일 HTML. 낮 장사(손님 유형 3종·아이템 10종·콤보) + 파견 1일 1건(모험가 풀 4명·텍스트 로그) + 7일 1주기·상납 1회. 목표: 코어 루프가 재밌는지 검증.

- v0.3: 메타(마정석 트리·전설 명부) + 분기 엔딩 3종.
- v1.0: 콘텐츠 확장(레시피 30+, 이벤트 60+), 밸런스, 사운드. 반응이 좋으면 Melvor Idle 경로(itch.io/Poki → Steam) 검토. [04]

## 8. 리스크와 대응
- **두 루프 중 한쪽만 파는 플레이** → 특별 주문(장사→파견 발주)과 재료 수급(파견→장사 공급)으로 상호 의존 강제. [Dave the Diver]
- **가격 게임의 정답 고착** → 트렌드+수요 하락+손님 유형 랜덤으로 정답을 계속 이동. [Moonlighter 교훈]
- **시간 압박 vs 코지 충돌** → 기본 난이도는 관대하게, "마왕 모드"(상납 가혹·발각 빠름)를 옵션으로. [CSD3 Zen, Potionomics 교훈]
- **후반 반복감** → 주차별 교단 의심 이벤트와 최종 분기가 매 런의 표정을 바꿈. [XCOM 2]
